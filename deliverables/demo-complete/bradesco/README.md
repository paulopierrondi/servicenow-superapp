# Bradesco Complete Demo

This HyperFrames project renders the complete Bradesco narrative with two personas:

- Executive user: opens the bank-like super app, asks the ServiceNow concierge for the day, reviews CMDB/incident/change evidence and approves the governed step.
- Fulfiller user: receives the same context, triages P1, validates CMDB Health and prepares ITSM/CSM/CRM actions.

Render from the repo root:

```bash
node scripts/render-complete-demo-videos.mjs
```

Output:

`deliverables/bradesco-demo/bradesco-servicenow-superapp-complete-demo.mp4`
